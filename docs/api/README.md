# API Documentation

A referência interativa e sempre atualizada da API é o **Swagger/OpenAPI** gerado pelo
FastAPI, disponível com a stack no ar em:

- Swagger UI: <http://localhost:8000/docs>
- OpenAPI JSON: <http://localhost:8000/openapi.json>

Para uma visão por router (auth, agents, channels, knowledge, settings, etc.), autenticação
JWT e settings dinâmicas, veja [docs/backend.md](../backend.md).

> Uma referência estática dedicada (exportada do OpenAPI) está **planejada**; por ora, use o
> Swagger acima como fonte de verdade.
